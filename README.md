# Local Economy Policy Optimization

This repository contains the decision-support framework developed for the paper: 
**"Multi-Objective Optimization for Efficient and Resilient Local Economy Design: An Industrial Engineering Approach"**.

## Project Structure
- `resilience.py`: Core optimization engine using PyGMO (NSGA-II).
- `outputs/`: Numerical results, including the full Pareto solution set.
- `figures/`: Visualizations of the trade-offs and scenario performance.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run simulation: `python resilience.py`
